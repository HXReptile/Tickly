# 项目名称（Tickly）

一个基于 Qt 开发的可视化高频率提醒工具。

---

## 项目简介

本项目用于提醒当日行为，比如说长期办公室，久坐。提醒自己该站起来活动、喝水等等行为。

---

## 功能特性

- 活动增删改  
- 顺序调整  
- 自启动
- 本地配置持久化（JSON）
